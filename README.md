# ?? DataForge: Automated Exploratory Data Analysis Engine

> *"Data speaks, but only if you know how to listen. DataForge translates the noise into insights."*

### ?? Overview
**DataForge** is a modular, interactive analytics platform designed to bridge the gap between raw datasets and actionable intelligence. Built with **Streamlit** and a custom **Python** backend, it automates the tedious parts of Exploratory Data Analysis (EDA)—cleaning, profiling, and visualizing—reducing the "Time-to-Insight" from hours to seconds.

### ??? Tech Stack
* **Core Engine:** Python 3.9+
* **Interactive UI:** Streamlit
* **Data Processing:** Pandas (Optimized for large datasets)
* **Visualization:** Plotly (Interactive), Seaborn (Statistical)
